Woocommerce Multiple Tabs plugin for WooCommerce products to show multiple tab. 

== Description ==

Woocommerce is great plugin allow you to add descriptions and review tabs by default
If you want to display more your product informations like Descriptions, Tech Specs, Kits, Tools, Accessories Woocommerce Multiple Tabs plugin will help you to show Tabs for your product

= Features =
* Multiple tabs
* Easy installation and manager your tabs
* Native WordPress Wysiwyg editor for your tab content
* Tab  ordering (priority) allow you to arrange your tab position
* Show or Hide tabs in your frontend easy



How to installation: http://joomquery.com/how-to-installation-woocommerce-multiple-tabs-plugin-for-wordpress
Author website: http://joomquery.com/
Support Plugin Page: http://joomquery.com/contact-us


== Installation ==

1. In your Dashboard click to add new plugin as normal Wordpress plugin installation in your search type "Woocommerce Multiple Tabs" and find Woocommerce Multiple Tabs plugin
Or you can download plugin from http://joomquery.com/how-to-installation-woocommerce-multiple-tabs-plugin-for-wordpress and then installation by Upload file
Or use your file manager or FTP to Upload the plugin to the '/wp-content/plugins/' directory. 
2. Activate it through the 'Plugins' menu in WordPress.
3. Enable plugin
